### 1. 기본 계산기 기능 정의

**필수 기능**
1. 기본 연산
   - 덧셈(+), 뺄셈(-), 곱셈(*), 나눗셈(/)
   - 괄호 연산
   - 백스페이스, AC(전체 삭제)

2. 입력/출력
   - 숫자 입력 (0-9)
   - 소수점 처리
   - 결과값 표시

3. 연산 기록
   - 이전 계산 내역 저장
   - 기록 확인 가능

### 2. 재미있는 계산기 아이디어 제안

1. **시간 여행 계산기** 🕰️
   - 과거/미래 날짜 계산
   - "오늘부터 100일 후는?", "1999일 전은?"
   - 기념일 계산, D-day 기능
   - 재미있는 시간 단위 변환 (예: "당신이 살아온 시간을 초로 환산하면?")

2. **감정 수학 계산기** 😊
   - 감정을 수치화해서 계산
   - "행복지수 8 + 스트레스지수 3 = 오늘의 컨디션"
   - 일기처럼 감정 기록을 남길 수 있음
   - 주간/월간 감정 그래프 제공

3. **음식 칼로리 계산기** 🍕
   - 음식 조합으로 칼로리 계산
   - "피자 2조각 + 콜라 1캔 - 30분 운동 = 남은 칼로리"
   - 식사 기록 저장
   - 운동 시간 추천

4. **프로그래머 계산기** 💻
   - 진수 변환 (2진수, 8진수, 16진수)
   - 비트 연산
   - 프로그래밍 언어별 데이터 타입 범위 계산
   - 코드 실행 시간 예측

5. **환경 영향 계산기** 🌍
   - 일상 활동의 탄소 발자국 계산
   - "자동차 10km + 에어컨 5시간 = CO2 배출량"
   - 환경 보호 팁 제공
   - 나무 심기로 환산

### 기능 상세 설계

1. **기본 계산 모드**
   - 일반적인 수학 연산 (+-*/)
   - 괄호, 소수점 지원
   - AC, 백스페이스
   - 계산 기록 저장

2. **감정 계산 모드**
   ```typescript
   type Emotion = {
     type: '기쁨' | '슬픔' | '화남' | '불안' | '스트레스' | '평온',
     value: number;  // 1-10 척도
     timestamp: Date;
     note?: string;  // 감정에 대한 메모
   }
   ```

   기능:
   - 감정 입력 (1-10 척도)
   - 감정 조합 계산
   - 일일 감정 요약
   - 감정 그래프 시각화

### UI 구성 제안

```
[모드 전환 버튼: 기본 ↔ 감정]

1. 기본 모드
+------------------------+
|       123 + 456       |  // 계산식 표시
|           = 579       |  // 결과 표시
+------------------------+
| AC  (  )   ←  |  ÷   |
| 7   8  9      |  ×   |
| 4   5  6      |  -   |
| 1   2  3      |  +   |
| 0   .  =      |  =   |
+------------------------+

2. 감정 모드
+------------------------+
|    오늘의 감정 입력    |
| 기쁨 (7) + 불안 (3)   |  // 감정 수식
|     = 평온 (5)        |  // 감정 결과
+------------------------+
| [감정 선택 버튼들]     |
| [강도 선택 (1-10)]    |
| [메모 입력창]         |
+------------------------+
| [감정 그래프 영역]     |
+------------------------+
```

### 특별한 기능들

1. **감정 연산 규칙**
   - 기쁨 + 기쁨 = 더 큰 기쁨
   - 기쁨 + 슬픔 = 복잡한 감정
   - 스트레스 - 평온 = 불안
   등의 재미있는 감정 조합 규칙 구현

2. **감정 분석**
   - 일일/주간/월간 감정 변화 그래프
   - 감정 패턴 분석
   - 긍정/부정 감정 비율

3. **추천 기능**
   - 현재 감정에 따른 음악 추천
   - 감정 개선을 위한 활동 추천
   - 감정 균형을 위한 조언
